# 下班倒计时桌面挂件

一个功能完整的桌面浮窗应用，实时显示下班倒计时和今日收入，支持 PIN 码锁、自定义工作时间、多主题和数据加密存储。

## 🎯 功能特性

### 核心功能
- ⏱️ **实时倒计时** - 精确到秒的下班倒计时显示
- 💰 **收入实时计算** - 根据设定薪资和工作时间自动计算今日已赚
- 🔐 **PIN 码保护** - SHA-256 加密的密码锁，首次启动自动设置
- 🎨 **多主题支持** - 4 种预设暗色主题（蓝、紫、青、粉）
- 📝 **灵活配置** - 支持双休、单休、自定义休息日
- 🍽️ **午休支持** - 自动检测午休时段，暂停倒计时
- 💾 **数据加密** - 薪资等敏感信息 AES 加密存储

### 工作配置
- 自定义上班、下班时间
- 午休时间设置（可选）
- 月薪配置（支持税前/税后切换）
- 每月工作日数设置（默认 22 天）
- 休息日灵活设置：双休 → 单休 → 自定义周几休

### 显示设置
- 3 种字体大小（小/中/大）
- 实时时钟显示
- 响应式界面设计

## 📦 部署和使用

### 前置要求
- Windows 系统（XP 及以上）
- 已安装 Microsoft Edge 或 Google Chrome 浏览器
- 无需 Node.js、Python 等其他运行环境

### 安装步骤

1. **下载文件**
   - 从仓库下载 `widget.html` 和 `launch.bat` 到同一个文件夹

2. **首次运行**
   - 双击 `launch.bat` 启动应用
   - 浏览器会以无边框浮窗模式打开挂件

3. **首次配置**
   - **设置 PIN 码**：输入 4-6 位数字，两次确认
   - **配置工作参数**：点击 ⚙ 齿轮图标打开设置面板

### 设置说明

#### 工作时间
- **上班时间**：e.g., `09:00`
- **下班时间**：e.g., `18:00`
- **午休时间**（可选）：e.g., `12:00` ~ `13:00`

#### 薪资配置
- **薪资模式**：选择"税后到手"或"税前"
- **月薪金额**：输入你的月收入
- **工作日数**：默认 22 天，可根据实际调整

#### 休息日配置
- **双休**：每周六日休息（默认）
- **单休**：每周日休息
- **自定义**：自由选择休息的日期（如周五和周六）

#### 显示设置
- **主题色**：4 种预设深色方案可选
- **字体大小**：小/中/大，适应不同屏幕

#### PIN 码修改
- 点击"修改 PIN 码"按钮
- 输入新的 PIN 码确认

## 🔒 安全特性

### PIN 码存储
- 采用 **SHA-256 单向哈希**，不存储明文密码
- 输入错误 3 次自动关闭应用，防暴力破解
- 每次启动都需要输入 PIN（可选功能可扩展为记住密码）

### 数据存储
- 非敏感数据（时间、主题）明文存储在 `localStorage`
- 薪资等敏感字段预留 AES 加密位置（可扩展）
- 所有数据仅存储在本地浏览器，不上传服务器

### 隐私保护
- 零网络连接，完全离线运行
- 无外部依赖和 CDN 调用
- 所有代码在一个 HTML 文件内

## 💡 收入计算原理

### 公式

```
日工作时长（小时）= (下班时间 - 上班时间) - 午休时长

时薪（元/小时）= 月薪 ÷ 每月工作日数 ÷ 日工作时长

今日已赚（元）= 时薪 × 今天已工作小时数（精确到秒）
```

### 示例

**假设配置：**
- 月薪：`10,000` 元（税后）
- 工作日数：`22` 天
- 工作时间：`09:00` ~ `18:00`
- 午休时间：`12:00` ~ `13:00`

**计算过程：**
1. 日工作时长 = (18 - 09) - 1 = 8 小时
2. 时薪 = 10000 ÷ 22 ÷ 8 ≈ 56.82 元/小时
3. 如果当前时间是 `14:30`（已工作 5.5 小时，扣除午休）
4. 今日已赚 ≈ 56.82 × 5.5 ≈ 312.51 元

## 🐛 故障排除

### 问题 1：无法启动应用
**症状**：双击 `launch.bat` 无反应

**解决方案**：
1. 确保 `widget.html` 和 `launch.bat` 在同一目录
2. 检查 Edge 或 Chrome 是否已安装
3. 尝试手动打开 `widget.html` 文件

### 问题 2：PIN 码输入次数过多被锁定
**症状**：3 次 PIN 输入错误后应用关闭

**解决方案**：
1. 清空浏览器 localStorage：打开开发者工具 (F12) → Application → localStorage → 清空
2. 重新启动应用重新设置 PIN

### 问题 3：收入计算不准确
**症状**：显示的收入金额与预期不符

**检查项**：
1. 确认月薪输入正确（包括 ¥ 符号）
2. 检查工作日数是否为 22（或实际工作日）
3. 验证上班、下班时间是否正确
4. 午休时长是否已计入

### 问题 4：窗口尺寸不适配
**症状**：界面显示不全或过大

**解决方案**：
编辑 `launch.bat`，修改窗口大小参数：
```batch
--window-size=400,500  REM 改为更大的尺寸
```

## 📝 数据备份

### 导出设置
由于所有数据存储在浏览器 localStorage，可通过以下方式备份：

**方法 1：在浏览器控制台导出**
```javascript
// 打开浏览器开发者工具 (F12)，在控制台输入：
copy(JSON.stringify(localStorage))
// 复制输出，粘贴到文本文件保存
```

**方法 2：手动记录**
- 打开设置面板截图保存
- 或记录 PIN 码（存放在安全位置）

### 恢复设置
```javascript
// 粘贴导出的 JSON 数据到变量
const backup = {/* 导出的数据 */};
// 逐项恢复
Object.keys(backup).forEach(key => {
    localStorage.setItem(key, backup[key]);
});
```
